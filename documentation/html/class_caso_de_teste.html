Descrição detalhada

Classe representativa dos casos de teste para os DOMINIOS.

Esta classe caracteriza os casos de teste. Cada caso terá:

  • Um código (CODIGO);
  • Um nome (TEXTO);
  • Uma data (DATA);
  • Uma ação (TEXTO);
  • Uma resposta (TEXTO); e
  • Um resultado (RESULTADO).

As restrições quanto ao formato dos membros são ditadas pelos tipos desses membros, como definidos em suas classes.
