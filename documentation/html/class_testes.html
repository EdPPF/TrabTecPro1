Descrição detalhada

Classe representativa da entidade de testes das classes de DOMINIOS.

Esta classe caracteriza os testes com um código (CODIGO), um texto (TEXTO) e uma classe (CLASSE).

As restrições quanto ao formato dos membros são ditadas pelos tipos desses membros, como definidos em suas classes.
